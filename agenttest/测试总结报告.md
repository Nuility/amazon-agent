### 失败处理

如果测试失败，请检查：

1. **环境检查失败**
   - 确认Python版本 >= 3.8
   - 检查项目目录结构

2. **依赖包检查失败**
   - 运行 `pip install -r requirements.txt`
   - 使用国内镜像源加速

3. **配置文件检查失败**
   - 确认config/config.yaml存在
   - 检查YAML格式正确性

4. **存储系统测试失败**
   - 检查data目录权限
   - 确保磁盘空间充足

5. **应用初始化失败**
   - 查看错误日志
   - 检查依赖注入配置

6. **用户服务测试失败**
   - 检查存储适配器
   - 验证用户仓储配置
